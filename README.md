使用方式:

```xml
<dependency>
    <groupId>YOUR.PROJECT.GROUPID</groupId>
    <artifactId>ARTIFACT-ID</artifactId>
    <version>VERSION</version>
</dependency>

<repository>
    <id>ARTIFACT-ID</id>
    <url>https://raw.github.com/chaimch/mvn-repo/mvn-repo/</url>
</repository>
```

部署方式:
```
mvn deploy
```
